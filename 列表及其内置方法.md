`start和stop可以为负数，但总是从左往右查找
凡是stop比start小，总是抛出ValueError`


```python
def index(lst, value, start = 0, stop= -1):
    i = start
    for x in lst[start: stop]:
        if x == value:
            return i
        i += 1
    raise ValueError()
```

```python
def count(lst, value):
    c = 0
    for x in lst:
        if x == value:
            c += 1
    return c
```

*  通过索引访问元素
*  index方法根据值返回第一个索引
*  count方法返回元素在列表出现的次数

#### index 和count的时间复杂度是O(n)线性复杂度 效率与数据规模线性相关

```python
def copy(lst):
    tmp = []
    for i in lst:
        tmp.append(i)
    return tmp
```
